Пошаговая настройка

Подробное руководство по установке и настройке эмуляторов на Steam Deck.

Способ 1: EmuDeck (рекомендуется)

EmuDeck — автоматический установщик, который настроит всё за вас.

  1. Включите Desktop Mode

    Зажмите кнопку питания → "Switch to Desktop".

  2. Скачайте EmuDeck

    Откройте браузер и перейдите на emudeck.com. Скачайте установщик.

  3. Запустите установку

    Найдите скачанный файл (обычно в Downloads) и запустите его двойным кликом.

    chmod +x EmuDeck.desktop
./EmuDeck.desktop
  4. Выберите конфигурацию

    Укажите, какие эмуляторы установить и куда сохранять ROM-файлы.

  5. Дождитесь завершения

    EmuDeck сам скачает и настроит все эмуляторы, RetroArch и Steam ROM Manager.

  6. Скопируйте ROM-файлы

    Папки для ROM-ов находятся в Emulation/roms на вашей SD-карте или SSD.

  7. Запустите Steam ROM Manager

    Добавьте игры в библиотеку Steam для запуска из Game Mode.

Способ 2: Вручную через Discover

Если вы хотите установить только один эмулятор без EmuDeck:

  1. Откройте Discover Store

    В Desktop Mode найдите Discover в меню приложений.

  2. Найдите эмулятор

    Поищите "Dolphin", "PCSX2", "PPSSPP" или "RetroArch".

  3. Установите

    Нажмите "Install" и дождитесь завершения.

  4. Настройте управление

    В настройках эмулятора выберите раскладку для геймпада Steam Deck.

Через Discover стоят стабильные версии. Для свежих сборок используйте Flatpak из бета-ветки.

Настройка управления

Контроллер Steam Deck

  • Все эмуляторы автоматически распознают контроллер Steam Deck
  • Для RetroArch используйте меню "Controls" для настройки кнопок
  • В Dolphin и PCSX2 контроллер настраивается в отдельном меню

Гироскоп

Гироскоп Steam Deck можно использовать для прицеливания в играх для Wii и PS3. Настройте его в параметрах эмулятора как эмуляцию мыши.

Трекпады

Трекпады отлично работают как эмуляция мыши для DOS-игр и стратегий. Используйте стандартную раскладку Steam Input.

Настройка производительности

  • TDP: Для PS2/GameCube выставьте 9-12 Вт; для PS1/SNES — 4-6 Вт
  • Разрешение: Используйте родное (1280×800) или 2x для PSP
  • VSync: Включите для устранения разрывов
  • Шейдеры: Включите кэширование для плавной игры
  • Частота кадров: Ограничьте до 60 FPS через настройки эмулятора
