Советы и хитрости

Проверенные советы для лучшего опыта эмуляции на Steam Deck.

Производительность

Настройки TDP и частоты

  • Для PS2/GameCube: TDP 9-11 Вт, частота GPU 1000-1200 МГц
  • Для PSP/PS1: TDP 4-6 Вт — экономит заряд батареи
  • Используйте Quick Access Menu (кнопка "…") для быстрой настройки

Кэширование шейдеров

В Dolphin и PCSX2 включите опцию "Compile Shaders Before Starting" (или аналог). Это уберёт микро-фризы во время игры.

В Yuzu/Ryujinx включите "Async Shader Compilation".

Разрешение

Для PSP используйте 2x-3x разрешение (960×544 или 1440×816). Для PS2 — 1x-2x (родное 640×480 или 1280×960).

Управление

Лучшие раскладки

  • PS2/PS1: L2/R2 на бамперы, L1/R1 на триггеры
  • GameCube: Назначьте аналоговые триггеры на курки (ZL/ZR)
  • N64: Используйте C-кнопки на правый стик
  • Wii: Гироскоп для указания, трекпады для кнопок

Сообщество раскладок

В Steam Community найдите раскладки управления для каждого эмулятора. Многие пользователи уже выложили оптимальные конфигурации.

Память и хранение

  • microSD: Используйте карты A2 (U3) для быстрой загрузки
  • Рекомендуемый размер: 256 ГБ — 1 ТБ
  • Форматируйте SD-карту в Steam Deck перед использованием
  • Переносите папку Emulation на SD-карту через EmuDeck

Работа с ROM-файлами

  • Используйте .chd для PS1/PS2/Sega — сжатие без потери качества
  • Для GameCube/Wii — .rvz формат (сжатие в Dolphin)
  • Для PSP — .iso или .cso (сжатый)
  • Сортируйте ROM-ы в папки по платформам
Файлы .chd и .rvz занимают на 30-60% меньше места!

Steam ROM Manager

Steam ROM Manager — утилита для добавления игр в библиотеку Steam с красивыми обоями.

  • Запускается из EmuDeck или отдельно
  • Автоматически парсит ROM-файлы и загружает арты
  • Добавляет игры в Steam с поддержкой Steam Cloud
  • Можно настроить исключение для отдельных платформ

После добавления игр, они появятся в Game Mode с обложками и описаниями.

Решение проблем

Игра вылетает

Проверьте: достаточно ли TDP, не перегревается ли Steam Deck, правильный ли регион BIOS.

Нет звука

В настройках эмулятора выберите ALSA или PulseAudio. Отключите аудио-буферизацию.

Тормозит

Понизьте разрешение, отключите тени и пост-эффекты, увеличьте TDP до 12-15 Вт.

Не сохраняется прогресс

Проверьте права на папки Emulation/saves. Запустите эмулятор из Desktop Mode один раз.
