BIOS и прошивки

Некоторые эмуляторы требуют файлы BIOS (прошивки) с оригинальных консолей. Эти файлы защищены авторским правом — вы должны получить их с собственной консоли.

Мы не предоставляем ссылки на скачивание BIOS-файлов. Извлеките их с вашей собственной консоли легальным способом.

Как извлечь BIOS

PlayStation 2 (PCSX2)

  1. Вставьте карту памяти PS2 в ПК (нужен адаптер)
  2. Используйте утилиту для дампа BIOS (например, dumpBIOS)
  3. Скопируйте файл .bin в папку Emulation/bios
  4. В PCSX2 укажите путь к BIOS

PlayStation 1 (RetroArch)

  1. Скачайте утилиту PSX BIOS dumper
  2. Запустите её на реальной PS1 или PS2
  3. Скопируйте SCPH*.bin файлы в папку RetroArch/system

Требуются файлы: scph5500.bin, scph5501.bin, scph5502.bin

PlayStation 3 (RPCS3)

  1. Установите последнюю прошивку PS3 через сайт PlayStation
  2. В RPCS3 выберите File → Install Firmware
  3. Укажите скачанный .PUP файл

Sega Saturn / Dreamcast

Для Saturn нужен BIOS-файл от региона вашей консоли. Для Dreamcast BIOS встроен в эмулятор Flycast.

Nintendo Wii U (Cemu)

Cemu требует ключи (keys.txt) для расшифровки игр. Их можно получить с вашей консоли Wii U, используя homebrew-приложения.

Проверка BIOS

В EmuDeck есть встроенная утилита проверки BIOS. Запустите EmuDeck → Tools → BIOS Checker, чтобы убедиться, что все файлы на месте.

Если BIOS отсутствует, эмулятор выдаст соответствующую ошибку при запуске игры.
